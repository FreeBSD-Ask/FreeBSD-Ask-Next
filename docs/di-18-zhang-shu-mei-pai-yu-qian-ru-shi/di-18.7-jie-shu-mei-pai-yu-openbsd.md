# 第 18.7 节 树莓派与 OpenBSD

OpenBSD 支持以下树莓派相关设备：

- Raspberry Pi 3
- Raspberry Pi 3 Model B+
- Raspberry Pi 4
- Raspberry Pi 400
- Raspberry Pi Compute Module 4

支持树莓派的以太网卡、WIFI、TF 卡、USB 3.0。
