# 第1.6节 UNIX 哲学


我认为 Linux 的道路走歪了，他把 Unix 哲学走成了苦难哲学，而且还在越走越歪，走了改旗易帜的邪路。单是 Linux Kernel 引入 Systemd 就能看出这一点。

Unix 哲学源于 UNIX 操作系统的开发，作者是肯·汤普森。Unix 哲学一言以蔽之即大道至简（`keep it simple, stupid`）：

 - 模块化：每次只用一个程序干一件事；
 - 清晰化：不要二进制；
 - 做好软件需求设计；
