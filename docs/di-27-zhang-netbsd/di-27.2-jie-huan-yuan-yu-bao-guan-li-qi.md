# 第 27.2 节 换源与包管理器


NetBSD 的包管理器是 pkgsrc，同时支持二进制与源代码编译安装。其二进制命令分别为 pkg_add 与 pkgin。

以 NetBSD 9.3 为例换二进制源：

```
echo http://mirrors.tuna.tsinghua.edu.cn/pkgsrc/packages/NetBSD/amd64/9.3/All > /usr/pkg/etc/pkgin/repositories.conf
```
