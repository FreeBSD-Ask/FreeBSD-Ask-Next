# 第 23.4 节 引导界面

## 调整引导选择界面的时间为 2 秒

```
# ee /boot/loader.conf
```

修改：

```
autoboot_delay="2"
```
## 精简启动输出

```
# echo boot_mute="YES"  >> /boot/loader.conf
```
