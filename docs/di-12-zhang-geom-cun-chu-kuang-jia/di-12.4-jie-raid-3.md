# 第 12.4 节 RAID 3

RAID 3：假设你有 N 块硬盘，其中 1 块硬盘作为校验使用，其余的 N-1 块硬盘相当于构成了 RAID 0，当其中一块硬盘损坏时，就可以通过校验硬盘的校验码还原出损坏磁盘的的原始数据。
