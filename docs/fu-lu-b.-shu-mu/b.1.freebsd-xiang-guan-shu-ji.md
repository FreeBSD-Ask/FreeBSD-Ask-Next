# B.1. 专门针对于 FreeBSD 的书籍

虽然手册的页面为 FreeBS D操作系统的每个部分都提供了确切的参考，但它们很少说明如何将这些部分联合起来，使整个操作系统顺利运行。在这方面，UNIX® 系统管理方面的好书或用户手册是无可替代的。
